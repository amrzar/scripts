### NAME

addheader -- add header to header [.h], code [.c], and assembler [.S] files

### SYNOPSIS

**addheader** -- template file --path path [--show] [list of files]

### DESCRIPTION

Update the license headers of files based on tempalte file. 

**--template**\
path to templete files that copied to target files

**--path**\
path to directory that contain header [.h], code [.c], and assembler [.S] files to be updated recursively

**--show**\
print initial lines of template files
